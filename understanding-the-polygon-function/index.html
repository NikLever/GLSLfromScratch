  • sides - number of sides in the polygon
  • radius
  • theta - angle between positive x axis and test point
  • index - which point on the polygon are we using for the test
  • rad - PI2/sides
  • d - length of vector from origin to test point, modulated
  • result - do we show this point

Notice that if you are inside the polygon the result is positive.
